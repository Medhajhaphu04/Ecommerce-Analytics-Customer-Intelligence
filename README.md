# AI-Powered E-Commerce Analytics & Customer Intelligence Platform

## Project Overview

This project leverages the Olist Brazilian E-Commerce Dataset to build an end-to-end analytics platform combining:

- SQL Analytics
- Customer Segmentation
- Sales Forecasting
- Customer Lifetime Value Prediction
- SHAP Explainability
- LLM-based Business Insights
- Streamlit Dashboard
